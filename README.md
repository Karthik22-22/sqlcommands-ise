# 🐬 MySQL Command Finder

A clean, searchable cheatsheet for MySQL commands — built for ISE students learning SQL.



---

## 📖 About

MySQL Command Finder is a single-page web app that lets you instantly search and find MySQL commands by keyword. No more Googling the same syntax over and over — just type what you're looking for and get the command.

---

## ✨ Features

- 🔍 **Instant search** — type any keyword and get matching commands
- 🏷️ **Category filters** — browse by CRUD, Filter, JOIN, Aggregate, Alter Table, Advanced
- 📋 **One-click copy** — copy any SQL command to clipboard instantly
- 📱 **Mobile responsive** — works great on phones and tablets
- 🌙 **Dark theme** — easy on the eyes

---

## 📚 Commands Included

| Category | Commands |
|----------|----------|
| CRUD | CREATE DATABASE, CREATE TABLE, INSERT, SELECT, UPDATE, DELETE |
| Filter | WHERE, LIKE, ORDER BY, LIMIT, DISTINCT, SHOW |
| JOIN | INNER JOIN, LEFT JOIN, RIGHT JOIN |
| Aggregate | COUNT / SUM / AVG, GROUP BY |
| Alter Table | ALTER TABLE, Constraints |
| Advanced | Subquery, Transactions, Views, Indexes, Export to CSV |
